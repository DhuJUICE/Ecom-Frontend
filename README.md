The React frontend for a food ordering and prepayment platform built to help small food vendors accept online card payments and reduce food wastage through pre-orders.

## Overview

This frontend provides the customer-facing interface for browsing a menu, placing orders, and completing secure online payments.  
It was designed for street food vendors who operate without card machines and often lose sales due to cash-only limitations.

## Key Features

- Product/menu browsing
- Cart functionality
- Checkout flow
- Integration with backend APIs
- Responsive, mobile-friendly layout
